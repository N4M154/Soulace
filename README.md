### Machine learning

- **Model:** Logistic Regression with `class_weight='balanced'` ensures fairness in predictions.
- **Preprocessing:** 
  - Label Encoding for categorical variables.
  - Standard Scaling for numerical data (e.g., age, sleep hours, and mental health scores).
- **Functionality:** Predicts the likelihood of needing professional mental health support based on user inputs.
- **Evaluation:** 
  - Stratified train-test splitting to maintain data balance.
  - Detailed classification reporting for accuracy.
- **Artifacts:** Trained model, scaler, and encoders are saved as `.pkl` files for seamless integration.

## ‚öô Getting Started

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/N4M154/Soulace.git
    ```

2. Install dependencies for the backend:
    ```bash
    cd api
    npm install
    ```
3. Install dependencies for the frontend:
    ```bash
    cd client
    npm install
    ```
4. Install dependencies for the root:
    ```bash
    cd ..
    npm install
    ```
5. Install dependencies for machine learning and AI:
   ```bash
   cd ml
   pip install -r requirements.txt
   ``` 

6. Add environment variables in the root `.env` file:
    ```bash
    MONGO = "<your_mongo_connection_string>"
    JWT_SECRET = "<your_jwt_secret>"
    CLOUD_NAME = "<your_cloudinary_cloud_name>"
    API_KEY = "<your_cloudinary_api_key>"
    API_SECRET = "<your_cloudinary_api_secret>"
    ```

7. Add environment variables in the client `.env` file:
    ```bash
    VITE_FIREBASE_API_KEY = "<your_firebase_api_key>"
    VITE_LASTFM_API_KEY = "<your_lastfm_api_key>"
    VITE_GROQ_API_KEY = "<your_groq_api_key>"
    VITE_NEWS_API_KEY = "<your_newsapi.org_api_key>"
    VITE_GOOGLE_UTUBE_API_KEY = "<your_youtube_data_api_v3_api_key>"
    ```
